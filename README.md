# MAXQDA HTML to Ecel
The aim of this project is to create a propper excel-file from the html output of the paraphrases.

The project does not need a database-server. It is able to work ith SQLite.

## stack
- Laravel 8
- spatie/simple-excel
- PHP 8.1
- SQLite
